PROVA DE HTML E PROGRAMÇÃO WEB

Afinal de contas, o que é HTML?

HTML significa HyperText Markup Language, ou Linguagem de marcação de Hipertexto. e é uma linguagem de marcação criada pra fazer documentos web.

Normalmente HTML é confundida como linguagem de programação, porém HTML não é uma linguagem de programação e sim de marcação.

Mas por que HTML não é considerada linguagem de programação? Pois sua principal função é definir estrutura e a apresentação visual de uma página web, ao invés de executar lógica ou algoritmos.

HTML utiliza tags para a formatação de informações, mas não possui capacidade de fluxo ou a capacidade de realizar cálculos complexos, que são características muito comuns entre as linguagens de programação.

Porém o HTML é comumente complementado com outra linguagem de programação pra montar estruturas, que é o caso do JavaScrpit, que de fato é uma linguagem de programação.

A linguagem de marcação HTML foi desenvolvida por Tim Berners-Lee, um físico britânico nos meados dos anos 90.

Diferente do que muitos pensam, o HTML não foi criado já perfeito e com toda estrutura pronta, na verdade ele já passou por diversas mudanças e atualizações ao longo dos anos.

A mais notável sendo a atualização para HTML5, onde trouxe consigo diversos novos elementos e recursos para melhorar no fim a experiência do usuário na WEB.

Com essa atualização também veio a modernização para suportar qualquer tipo de HTML em qualquer navegador, que são leitores de HTML.
